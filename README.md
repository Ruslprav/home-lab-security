# 🛡️ Home-Lab Infrastructure & Security Hardening

### 📋 Описание проекта
Развертывание и базовая защита домашней виртуальной ИТ-инфраструктуры. 
Цель проекта — создание изолированной среды для тестирования средств защиты информации (СЗИ).

---

### 🌐 Архитектура стенда
Стенд развернут на базе гипервизора **VMware / VirtualBox** в изолированном сетевом сегменте.

*   **Node 1 (Server):** Ubuntu Server 22.04 LTS 
*   **Node 2 (Client):** Windows 10 Pro 
*   **Network:** Подсеть `192.168.56.0/24` (Host-only Adapter)

---

### 🛠 Реализованные меры безопасности (Hardening)

#### 1. Сетевая настройка
*   Установлены **статические IP-адреса** для всех узлов.
*   Проверена связность узлов (ICMP/Ping) внутри сегмента.

#### 2. Минимизация поверхности атаки
*   Проведен аудит открытых портов (`netstat / ss`).
*   Отключены неиспользуемые системные службы и демоны.

#### 3. Защита удаленного доступа (SSH)
*   **Запрещен** вход под пользователем `root`.
*   Настроена аутентификация строго по **SSH-ключам** (RSA 4096).
*   Изменен стандартный порт 22 на кастомный.

#### 4. Межсетевой экран (Firewall)
*   Сконфигурирован **UFW** (Uncomplicated Firewall).
*   Политика по умолчанию: `Deny Incoming` (Запретить всё входящее).
*   Разрешены только необходимые порты: HTTP (80), HTTPS (443).
