🛡️ Home-Lab Infrastructure & Security Hardening
📋 Описание проекта
Развертывание и базовая защита домашней виртуальной ИТ-инфраструктуры. Цель проекта — создание изолированной среды для тестирования средств защиты информации (СЗИ) и отработки навыков системного администрирования.
🌐 Архитектура стенда
Стенд развернут на базе гипервизора VMware Workstation / VirtualBox в изолированном сетевом сегменте (Host-only Adapter).
Node 1 (Server): Ubuntu Server 22.04 LTS (Роль: Веб-сервер/Хранилище).
Node 2 (Client): Windows 10 Pro (Роль: Рабочее место администратора).
Network: Подсеть 192.168.56.0/24.
🛠 Реализованные меры безопасности (Hardening)
В рамках проекта были выполнены следующие шаги по укреплению защиты (Hardening):
Сетевая связность:
Настроены статические IP-адреса для исключения конфликтов в сети.
Настроено разрешение имен (DNS/Hosts) внутри сегмента.
Минимизация поверхности атаки:
Проведен аудит открытых портов утилитой netstat.
Отключены неиспользуемые системные службы и демоны.
Защита удаленного доступа (SSH):
Отключен вход под пользователем root.
Настроена аутентификация строго по SSH-ключам (RSA 4096).
Изменен стандартный порт 22 на кастомный для защиты от автоматизированных сканеров.
Межсетевой экран (Firewall):
Сконфигурирован UFW (Uncomplicated Firewall).
Политика по умолчанию: Deny Incoming, Allow Outgoing.
Разрешены только необходимые порты (HTTP, HTTPS, кастомный SSH).
